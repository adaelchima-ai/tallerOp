Bienvenido

Utilice el menú superior para navegar entre las funcionalidades:

  • Gestión de Dueños: Para registrar y administrar la información de los propietarios (R1).
  • Gestión de Mascotas: Para registrar y administrar las mascotas, asignandoles un dueño registrado (R2).
  • Listado Mascotas-Dueños: Para ver una tabla resumen de todas las mascotas con su respectivo dueño (R3).
